# DouDiZhu_with_AI
一个命令行显示的斗地主游戏，每天直播更新。

一期工程： 斗地主游戏（命令行）【2week】
二期工程：强化学习设置 【2week以上？】

一期游戏逻辑：
- 步骤： 发牌（30min ok），游戏循环（抢地主35min ok，出牌 ok，验证 ok，胜利结算（ok）
- 数据结构：玩家（Player），游戏（Game），规则检查器（Comparer）

一局的得分要尽量细化，方便训练AI。
（比如胜多少，赢到什么程度）
